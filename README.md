# emacs-config
My personal Emacs configs, consisting mostly of configuration of free
tools.

INSTALL

Usually, I sync this in /home/$USER/Emacs and then have ~/.emacs
consisting of ths single line:

  (load-file "~/Emacs/personal.el")

EXPECTED PACKAGES
